Exercice 1

 La balise html : Définit le début et la fin d'un document HTML.
La balise head : Contient les informations de la page HTML (comme le titre, les liens vers les feuilles de style, etc.).
La balise title : Définit le titre de la page qui apparaît dans l'onglet du navigateur.
La balise meta : Fournit des métadonnées comme l'encodage, la description, etc.
La balise link : Utilisé pour lier des ressources externes, comme les fichiers CSS.
La balise style : Contient les styles CSS en ligne pour la page.
La balise body : Contient le contenu visible de la page (texte, images, etc.).
La balise h1 à h6 : Balises de titre, h1 étant le plus important et h6 le moins important.
La balise p : Définit un paragraphe de texte.
La balise a : Crée un lien hypertexte.
La balise img : Insère une image.
La balise ul : Crée une liste non ordonnée.
La balise ol : Crée une liste ordonnée (numérotée).
La balise li : Définit un élément de liste dans une liste ul ou ol.
La balise div : Balise de conteneur générique pour regrouper des éléments et appliquer du style ou de la structure.
La balise span : Balise en ligne pour appliquer un style ou une structure sur un petit bloc de texte.
La balise form : Définit un formulaire.
La balise input : Crée un champ de saisie pour un formulaire (texte, bouton, case à cocher, etc.).
La balise button : Crée un bouton interactif.
La balise table : Crée un tableau.
La balise tr : Définit une ligne dans un tableau.
La balise td : Définit une cellule de données dans une ligne de tableau.
La balise th : Définit une cellule d'en-tête dans une ligne de tableau.
La balise br : Crée un saut de ligne.
La balise hr : Crée une ligne horizontale pour séparer des sections.
La balise strong : Met en valeur un texte en gras.
La balise em : Met en valeur un texte en italique.
La balise u : Souligne le texte.
La balise mark : Surligne le texte.
La balise code : Met la police code sur la partie HTML qu'on veut.
La balise iframe : Permet d'intégrer un document externe (comme une vidéo ou une carte).
La balise script : Permet d'inclure du JavaScript dans la page.
La balise footer : Définit le pied de page du document.
